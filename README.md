# The Amazing Tales


## Features

Connect world-wide to share your stories.


### Project Setup

1. Clone the repository

```bash
git clone https://github.com/UtkJaiswal/the_amazing_tales.git
```

2. Change the directory

```bash
cd the_amazing_tales
```

3. Install the dependencies

```bash
npm install
```

4. Run the project

```bash
npm start
```
